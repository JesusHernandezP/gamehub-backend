# 🎮 GameHub Backend

Backend REST desarrollado con Java y Spring Boot para una plataforma de videojuegos orientada a gestión de usuarios, autenticación y administración de torneos.

El proyecto fue construido aplicando arquitectura por capas, seguridad con JWT y persistencia relacional utilizando PostgreSQL.

---

# 🚀 Objetivo

Desarrollar una API backend moderna y escalable enfocada en:

- autenticación segura
- gestión de usuarios
- administración de torneos
- persistencia de datos
- arquitectura backend mantenible

---

# 🛠 Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- JPA / Hibernate
- Maven
- Docker
- Swagger / OpenAPI

---

# ✨ Características principales

- API REST
- Autenticación con JWT
- Arquitectura por capas
- Gestión de usuarios
- Gestión de torneos
- Persistencia relacional
- Documentación Swagger
- Contenedorización con Docker

---

# 📚 Arquitectura

El proyecto sigue una estructura backend modular utilizando separación por capas para controladores, servicios, persistencia y seguridad.

# 📖 Documentación API

Swagger UI disponible para pruebas e inspección de endpoints.
